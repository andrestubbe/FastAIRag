TITLE:
Making the JVM Faster: Native C++ Vector Search & Local 1B RAG Pipeline in Java! 🚀

DESCRIPTION:
Introducing FastAIVectorDB & FastAIRag — high-performance, native Windows vector storage and retrieval-augmented generation primitives for the FastJava AI ecosystem. 

FastAIVectorDB brings raw C++ execution speed to Java vector searches via optimized JNI bindings, featuring a complete thread-safe Java fallback. FastAIRag couples this speed with local models like Llama 3.2 1B to deliver instant command-line query assistance with zero framework bloat.

KEY HIGHLIGHTS:
- [FastAIVectorDB]: Raw C++ JNI implementation of Cosine Similarity linear scan (AVX/HNSW planned).
- [FastAIRag]: Ultra-lightweight RAG orchestration (Modular Chunking, EmbeddingProvider interfaces).
- [Pal Demo]: Instant Windows CMD CLI helper utilizing local Llama 3.2 1B via Ollama.
- [FastTerminal integration]: Smooth, custom monochrome █ progress indicator during offline index creation.

RESULTS:
- Standard Java Fallback Scan: ~5 ms
- Native C++ JNI Scan: <1 ms
- Response latency: Instantly streamed token-by-token directly in the console.
